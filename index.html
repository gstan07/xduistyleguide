Tooltips and pop-overs



Tooltips: Show/hide tooltip custom attributes you can use 
data-component-tooltip-animation, default:false, apply a css fade transition to the tooltip
data-component-tooltip-placement, default:'top', how to position the tooltip - top | bottom | left | right
data-component-tooltip-selector, default:false, If a selector is provided, tooltip objects will be delegated to the specified targets.
data-component-tooltip-title, default: "", default title value if `title` tag isn't present
data-component-tooltip-trigger, default:'hover', how tooltip is triggered - hover | focus | manual
data-component-tooltip-delay, default:0, delay showing and hiding the tooltip (ms) - does not apply to manual trigger type If a number is supplied, delay is applied to both hide/show Object structure is: delay: { show: 500, hide: 100 }

data-component-spinner-length, default: 5, // The length of each line
data-component-spinner-width, default: 2, // The line thickness
data-component-spinner-radius, default: 7, // The radius of the inner circle
data-component-spinner-corners, default: 1, // Corner roundness (0..1)
data-component-spinner-rotate, default: 0, // The rotation offset
data-component-spinner-color, default: '#000', // #rgb or #rrggbb
data-component-spinner-speed, default: 1, // Rounds per second
data-component-spinner-trail, default: 60, // Afterglow percentage
data-component-spinner-shadow, default: false, // Whether to render a shadow
data-component-spinner-hwaccell, default: false, // Whether to use hardware acceleration
data-component-spinner-classname, default: 'spinner', // The CSS class to assign to the spinner
data-component-spinner-zIndex, default: 2e9, // The z-index (defaults to 2000000000)
data-component-spinner-top, default: 10, // Top position relative to parent in px
